# 🎬 CineMatch AI

An explainable AI-powered movie recommendation system built using FastAPI, React, TypeScript, and Scikit-Learn.

CineMatch AI recommends similar movies based on content analysis using TF-IDF Vectorization and Cosine Similarity.

---

## 🚀 Features

- Movie recommendation engine
- TF-IDF based content filtering
- Cosine similarity matching
- FastAPI REST APIs
- React + TypeScript frontend
- Responsive recommendation cards
- TMDB 5000 movie dataset integration
- Error handling and loading states

## ⚙️ How It Works

1. User enters a movie title.
2. React frontend sends a request to FastAPI.
3. FastAPI calls the recommendation engine.
4. TF-IDF vectorization converts movie descriptions into vectors.
5. Cosine similarity finds the most similar movies.
6. Recommendations are returned to the frontend.
